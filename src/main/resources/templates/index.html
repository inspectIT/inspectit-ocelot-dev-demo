Welcome!

Welcome to Novatec's EUM-Server training! This page provides you with a short description of what you can find here.

HTML Editor: You will find a small Text-Editor which you can use to edit both the greeting and the form data page.
Greeting: On this page you can find an image and a button. When the button is pushed, a new image will be loaded.
Form Data: This page contains two text fields. Enter a text in the left field and submit it with the submit button. Doing so will send a request to the server which will return your initial text in caps.
EUM Configuration: Use this page to control the EUM-Server. You can start, stop and restart the server aswell as see if the server is currently running. Use the YAML-Editor on this page to edit the eum server's configuration.

Stuck?

Feel free to ask our coaches or check the EUM-Server's documentation!
